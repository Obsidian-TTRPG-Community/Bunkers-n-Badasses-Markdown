# Wyvern Familiar
___Requires:___ [Clawbringer](Clawbringer)
You gain a Wyvern Familiar that deals `dice: 1d6`(1d6) Incendiary Damage to any Target it flies past.
If it dies, it respawns at the end of the Encounter.
